pok plugin for `Tutor <https://docs.tutor.edly.io>`__
#####################################################

Tutor plugin to integrate POK certificates into Open edX


Installation
************

.. code-block:: bash

    pip install git+https://github.com/aulasneo/tutor-contrib-pok

Usage
*****

.. code-block:: bash

    tutor plugins enable pok
